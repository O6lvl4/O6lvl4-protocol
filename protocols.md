# Protocols

[README](./README.md) で定義した運用ルールを、カテゴリごとに厳密定義したプロトコル本体。各プロトコルは固定構造で記述する：**目的 / 不変条件 / パラメータ / 実行手順 / 違反検知 / 復旧手順**。

優先順位は P1 → P6 → P4 → P5 → P9 → P2 → P7 → P8 → P3 → P10。

---

## P1. 睡眠プロトコル

**目的**: 翌朝の脳の起動品質を最大化する。

**不変条件**

- 睡眠時間 ≥ 7時間
- 就寝時刻のずれ ≤ ±15分
- 起床時刻のずれ ≤ ±30分（週末含め±60分まで許容）

**パラメータ**

- 就寝: 23:30
- 起床: 07:00
- 寝室温度: 18〜20℃
- 寝室照度: 入眠時 ≤ 1lux
- 入眠目標: 床入り後20分以内

**実行手順**

1. 22:00 ブルーライト遮断、室温調整
2. 22:30 強い思考タスク終了、シャワーは就寝90分前
3. 23:00 部屋を暗くする、紙の本のみ可
4. 23:30 就寝
5. 07:00 起床、即カーテンを開ける

**違反検知**

- 入眠潜時 > 30分が3日連続
- 中途覚醒 ≥ 2回が3日連続
- 起床時 HRV がベースライン-20%以下

**復旧手順**

1. アルコール・夜の運動・夜の仕事を48時間停止
2. カフェインを12時以降ゼロに変更
3. 改善しなければ P6 を見直し

## P2. 有酸素プロトコル

**目的**: 脳への酸素・血流供給能を上げ、ミトコンドリア機能を維持する。

**不変条件**

- 1日の総歩数 ≥ 7,000歩
- 連続座位 ≤ 90分
- 高強度（Zone 4 以上）は週 ≤ 2回、間に48時間以上空ける
- 完全静止日 ≤ 週1日

**パラメータ**

- Zone 2: 心拍最大の60〜70%、会話可能、毎日30〜60分
- Zone 4〜5: 心拍最大の85%以上、週0〜2回、合計20分以内
- 空腹時の高強度は禁止

**強度の週次配分**

| 種類 | 頻度 | 1回時間 |
|---|---|---|
| Zone 2 | 週6〜7回 | 30〜60分 |
| Zone 3 | 週0〜1回 | 30分 |
| Zone 4〜5（インターバル） | 週0〜2回 | 15〜20分 |
| 完全休養 | 週0〜1回 | - |

**実行手順（毎日）**

1. 朝 or 昼に Zone 2 を30〜60分（速歩・バイク・軽ジョグから選択）
2. 90分ごとに2〜5分立つ・歩く
3. 高強度日は別枠で実施し、その日は筋トレを入れない

**違反検知**

- 起床時心拍 ≥ ベースライン+10bpm が3日連続
- 午前の集中時間が前週比-30%以上

**復旧手順**

1. Zone 4 以上を1週間停止、Zone 2 のみに切り替え
2. 改善しなければ強度を1段下げて2週間維持

## P3. 筋力プロトコル

**目的**: 姿勢保持・代謝・骨密度を支えるインフラ筋を維持する。**筋肥大は目的にしない**。

**不変条件**

- 週2〜3回、各40分以内
- 限界の2レップ手前で止める（RPE ≤ 8）
- 同一部位は48時間以上空ける
- 知的労働の前に高ボリューム筋トレを入れない

**パラメータ**

- セット数: 各種目2〜3セット
- レップ数: 6〜12
- セット間休息: 90〜120秒
- 種目数: 1セッション4〜5種目

**種目セット**

- A日（押す＋下半身）: スクワット / ベンチ or 腕立て / ショルダープレス / プランク
- B日（引く＋ヒンジ＋体幹）: デッドリフト軽め / 懸垂 or 斜め懸垂 / ローイング / デッドバグ

**実行手順**

1. ウォームアップ5分（軽い有酸素＋関節モビリティ）
2. メイン種目から重い順に実施
3. 各セット、フォーム崩壊を感じたら即終了
4. クールダウン3分（呼吸・ストレッチ）

**違反検知**

- 翌日の集中時間が著しく低下
- 関節痛・腰痛の発生
- 同重量で2週連続レップ数低下

**復旧手順**

1. 1週間ボリューム50%カット
2. 痛み発生時はその種目を2週間停止し代替種目へ

## P4. 食事プロトコル

**目的**: 血糖を安定させ、午後の認知低下を防ぐ。

**不変条件**

- 1日3食、固定時刻 ±30分
- 毎食タンパク質 ≥ 20g
- 昼食の炭水化物 ≤ 朝・夜の半量
- 就寝3時間前以降の固形物摂取禁止
- 加工糖含有飲料は完全禁止

**パラメータ**

- タンパク質: 体重(kg) × 1.6g/日
- 食事時刻: 朝07:30、昼12:30、夜19:00
- 食事所要時間: ≥ 15分（早食い禁止）

**実行手順**

- **朝**: タンパク質30g + 低 GI 炭水化物 + 脂質。例: 卵2個 + 全粒パン + ヨーグルト
- **昼**: タンパク質中心、丼物・麺単体禁止。例: 魚 or 鶏 + 野菜 + 少量の米
- **夜**: タンパク質 + 野菜 + 主食。最も量を取ってよい時間帯
- **間食**: 必要時のみ、ナッツ・果物・ヨーグルト

**違反検知**

- 食後60〜120分の眠気・集中低下
- ウエストが目標+2cm 超
- 空腹による集中崩壊が週3回以上

**復旧手順**

1. 食後眠気 → 次回その食事の炭水化物を半減
2. ウエスト超過 → 昼食の炭水化物のみ削減
3. 空腹崩壊 → 朝食のタンパク質と脂質を増量

## P5. 水分・カフェインプロトコル

**目的**: 脱水による認知低下を防ぎ、カフェインを「使う」状態を維持する（「依存される」状態を避ける）。

**不変条件**

- 1日の水分摂取 ≥ 体重(kg) × 30ml
- カフェイン総量 ≤ 400mg/日
- 14:00以降のカフェイン摂取禁止
- 起床直後60分はカフェイン禁止

**パラメータ**

- 起床直後: 水500ml
- 各食事: 水200〜300ml
- 作業中: 1時間ごとに200ml
- カフェイン: 09:30〜13:30の間で最大2杯

**実行手順**

1. 起床 → 水500ml → 90分後にコーヒー1杯
2. 昼食前後にコーヒー1杯（必要時のみ）
3. 14:00以降は水・白湯・カフェインレスのみ

**違反検知**

- 14時以降の急激な眠気 → 水分不足の可能性
- 朝のカフェインなしで動けない → 依存兆候
- 入眠潜時の悪化 → カフェイン残留

**復旧手順**

1. 依存兆候 → 週末2日カフェイン完全停止
2. 入眠悪化 → カットオフを12:00に前倒し

## P6. 光・概日リズムプロトコル

**目的**: 体内時計を毎日リセットし、睡眠・覚醒・ホルモン分泌を安定させる。

**不変条件**

- 起床後60分以内に2,000lux 以上の光を浴びる
- 日没後はリビング照度 ≤ 100lux
- 就寝60分前から青色光遮断
- 起床時刻の固定（P1 と連動）

**パラメータ**

- 朝の光曝露: ≥ 10分（屋外）または ≥ 20分（窓際）
- 日中の光曝露: 累計 ≥ 30分（屋外）
- 夜の照明: 暖色（2700K 以下）

**実行手順**

1. 起床直後にカーテン全開、可能なら屋外散歩10分
2. 昼休みに5〜15分屋外
3. 18:00以降に間接照明へ切り替え
4. 21:00以降スクリーンはダーク＋ナイトシフト

**違反検知**

- 入眠困難の慢性化
- 午前のだるさ
- 週末の極端な睡眠時刻シフト

**復旧手順**

1. 朝の屋外光曝露を3日連続で実施
2. 夜のスクリーン使用時間を1時間短縮

## P7. 座位・姿勢プロトコル

**目的**: 血流・呼吸・脳灌流を維持し、慢性疲労と腰痛を防ぐ。

**不変条件**

- 連続座位 ≤ 90分
- 1日の立位/歩行時間 ≥ 累計2時間
- モニター上端を目線高さに
- キーボード時の手首角度 ±15度以内

**パラメータ**

- 立ち上がり頻度: 60〜90分ごと
- 立ち上がり時間: 2〜5分
- スタンディング時間: 1日のうち累計60〜120分

**実行手順**

1. タイマーを60分にセット
2. 鳴ったら立つ → 水を飲む → 2分歩く or ストレッチ
3. 午後の作業のうち少なくとも1セッションをスタンディングで実施

**違反検知**

- 夕方の腰痛・首こり
- 午後の集中力急落
- 1日の歩数 < 5,000歩

**復旧手順**

1. タイマーを45分に短縮
2. 1日3回の5分散歩を強制挿入

## P8. 認知ワークプロトコル

**目的**: 1日に深い集中時間を最大化する。

**不変条件**

- 最重要タスクは午前の同じ時間帯に固定
- ディープワーク中は通知全遮断
- 1セッション ≤ 90分、間に ≥ 15分の休憩
- ディープワーク中の意思決定は仕事のみ（昼食何にするか等は前夜に決定）

**パラメータ**

- ディープワーク帯1: 09:00〜12:00
- ディープワーク帯2: 13:00〜15:30
- セッション長: 50〜90分
- 休憩: 10〜20分（散歩・ストレッチ・水分）

**実行手順**

1. 前夜に翌日の最重要タスク1つを決定
2. 朝のシャローワークは09:00で打ち切る
3. ディープワーク中は Slack・メール・SNS をアプリ単位でブロック
4. 休憩中は意図的に画面を見ない

**違反検知**

- 1日の深い集中時間 < 3時間が週3日以上
- 会議・通知でディープワーク帯が侵食された日が週2日以上

**復旧手順**

1. 会議は午後に集約、午前ブロックを死守
2. 通知遮断ツールを強制化

## P9. 回復プロトコル

**目的**: 慢性疲労を蓄積させず、長期的な認知パフォーマンスを保つ。

**不変条件**

- 完全な仕事オフ日 ≥ 週1日（思考タスクなし）
- 連続労働日数 ≤ 6日
- 月1回の1日完全静養日

**パラメータ**

- 平日の労働時間: ≤ 10時間
- 週末の労働時間: ≤ 4時間（理想0）
- 休暇: 四半期ごとに3〜5日

**実行手順**

1. 日曜日は仕事の Slack・メールを開かない
2. 週次レビューは金曜午後に実施し、月曜の不安を消す
3. 疲労兆候が出たら計画的に半休を取る

**違反検知**

- 朝の HRV が2週連続でベースライン-15%
- 休日に休んだ感覚が得られない
- 週次レビューで「集中できた時間」が3週連続減少

**復旧手順**

1. 即座に2日連続オフを確保
2. 運動強度を1週間半減
3. 改善しなければ睡眠を9時間に延長して1週間

## P10. 計測プロトコル

**目的**: 最小コストでシステムの異常を検知する。

**不変条件**

- 計測項目を増やさない（決めた指標のみ）
- 毎朝同条件で測定（起床直後・トイレ後・服装統一）
- 単発の値で判断せず7日移動平均で見る

**パラメータ**

| 指標 | 頻度 | 用途 |
|---|---|---|
| 体重 | 週3回・朝 | 体組成トレンド |
| 安静時心拍 / HRV | 毎朝 | 回復状態 |
| 睡眠時間 | 毎日 | 基盤指標 |
| 自己評価3項目（睡眠の質・気分・集中見込み） | 毎朝 | 主観統合 |
| 深い集中時間 | 毎日 | 唯一の成果指標 |
| ウエスト | 月1回 | 体組成検証 |

**実行手順**

1. 朝: 体重・心拍・自己評価を1分以内で記録
2. 夜: その日の深い集中時間を記録
3. 週末: 移動平均をチェックし、各プロトコルへフィードバック

**違反検知**

- 「深い集中時間」7日平均が前週比-20%
- 主観スコアの和が3日連続で平均以下

**復旧手順**

- 該当する上位プロトコル（P1〜P9）の違反検知を順にチェック
- 最初に該当したものを24時間以内に修正
