# P05. 水分・カフェインプロトコル

**目的**: 脱水による認知低下を防ぎ、カフェインを「使う」状態を維持する（「依存される」状態を避ける）。

## 不変条件

- 1日の水分摂取 ≥ 体重(kg) × 30ml
- カフェイン総量 ≤ 400mg/日
- 14:00以降のカフェイン摂取禁止
- 起床直後60分はカフェイン禁止

## パラメータ

- 起床直後: 水500ml
- 各食事: 水200〜300ml
- 作業中: 1時間ごとに200ml
- カフェイン: 09:30〜13:30の間で最大2杯

## 実行手順

1. 起床 → 水500ml → 90分後にコーヒー1杯
2. 昼食前後にコーヒー1杯（必要時のみ）
3. 14:00以降は水・白湯・カフェインレスのみ

## 違反検知

- 14時以降の急激な眠気 → 水分不足の可能性
- 朝のカフェインなしで動けない → 依存兆候
- 入眠潜時の悪化 → カフェイン残留

## 復旧手順

1. 依存兆候 → 週末2日カフェイン完全停止
2. 入眠悪化 → カットオフを12:00に前倒し
