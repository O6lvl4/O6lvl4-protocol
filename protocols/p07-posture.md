# P07. 座位・姿勢プロトコル

**目的**: 血流・呼吸・脳灌流を維持し、慢性疲労と腰痛を防ぐ。

## 不変条件

- 連続座位 ≤ 90分
- 1日の立位/歩行時間 ≥ 累計2時間
- モニター上端を目線高さに
- キーボード時の手首角度 ±15度以内

## パラメータ

- 立ち上がり頻度: 60〜90分ごと
- 立ち上がり時間: 2〜5分
- スタンディング時間: 1日のうち累計60〜120分

## 実行手順

1. タイマーを60分にセット
2. 鳴ったら立つ → 水を飲む → 2分歩く or ストレッチ
3. 午後の作業のうち少なくとも1セッションをスタンディングで実施

## 違反検知

- 夕方の腰痛・首こり
- 午後の集中力急落
- 1日の歩数 < 5,000歩

## 復旧手順

1. タイマーを45分に短縮
2. 1日3回の5分散歩を強制挿入
