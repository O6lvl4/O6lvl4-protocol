# P4. 食事プロトコル

**目的**: 血糖を安定させ、午後の認知低下を防ぐ。

## 不変条件

- 1日3食、固定時刻 ±30分
- 毎食タンパク質 ≥ 20g
- 昼食の炭水化物 ≤ 朝・夜の半量
- 就寝3時間前以降の固形物摂取禁止
- 加工糖含有飲料は完全禁止

## パラメータ

- タンパク質: 体重(kg) × 1.6g/日
- 食事時刻: 朝07:30、昼12:30、夜19:00
- 食事所要時間: ≥ 15分（早食い禁止）

## 実行手順

- **朝**: タンパク質30g + 低 GI 炭水化物 + 脂質。例: 卵2個 + 全粒パン + ヨーグルト
- **昼**: タンパク質中心、丼物・麺単体禁止。例: 魚 or 鶏 + 野菜 + 少量の米
- **夜**: タンパク質 + 野菜 + 主食。最も量を取ってよい時間帯
- **間食**: 必要時のみ、ナッツ・果物・ヨーグルト

## 違反検知

- 食後60〜120分の眠気・集中低下
- ウエストが目標+2cm 超
- 空腹による集中崩壊が週3回以上

## 復旧手順

1. 食後眠気 → 次回その食事の炭水化物を半減
2. ウエスト超過 → 昼食の炭水化物のみ削減
3. 空腹崩壊 → 朝食のタンパク質と脂質を増量
