# P10. 計測プロトコル

**目的**: 最小コストでシステムの異常を検知する。

## 不変条件

- 計測項目を増やさない（決めた指標のみ）
- 毎朝同条件で測定（起床直後・トイレ後・服装統一）
- 単発の値で判断せず7日移動平均で見る

## パラメータ

| 指標 | 頻度 | 用途 |
|---|---|---|
| 体重 | 週3回・朝 | 体組成トレンド |
| 安静時心拍 / HRV | 毎朝 | 回復状態 |
| 睡眠時間 | 毎日 | 基盤指標 |
| 自己評価3項目（睡眠の質・気分・集中見込み） | 毎朝 | 主観統合 |
| 深い集中時間 | 毎日 | 唯一の成果指標 |
| ウエスト | 月1回 | 体組成検証 |

## 実行手順

1. 朝: 体重・心拍・自己評価を1分以内で記録
2. 夜: その日の深い集中時間を記録
3. 週末: 移動平均をチェックし、各プロトコルへフィードバック

## 違反検知

- 「深い集中時間」7日平均が前週比-20%
- 主観スコアの和が3日連続で平均以下

## 復旧手順

- 該当する上位プロトコル（[P1](./p01-sleep.md)〜[P9](./p09-recovery.md)）の違反検知を順にチェック
- 最初に該当したものを24時間以内に修正
