# P01. 睡眠プロトコル

**目的**: 翌朝の脳の起動品質を最大化する。

## 不変条件

- 睡眠時間 ≥ 7時間
- 就寝時刻のずれ ≤ ±15分
- 起床時刻のずれ ≤ ±30分（週末含め±60分まで許容）

## パラメータ

- 就寝: 23:30
- 起床: 07:00
- 寝室温度: 18〜20℃
- 寝室照度: 入眠時 ≤ 1lux
- 入眠目標: 床入り後20分以内

## 実行手順

1. 22:00 ブルーライト遮断、室温調整
2. 22:30 強い思考タスク終了、シャワーは就寝90分前
3. 23:00 部屋を暗くする、紙の本のみ可
4. 23:30 就寝
5. 07:00 起床、即カーテンを開ける

## 違反検知

- 入眠潜時 > 30分が3日連続
- 中途覚醒 ≥ 2回が3日連続
- 起床時 HRV がベースライン-20%以下

## 復旧手順

1. アルコール・夜の運動・夜の仕事を48時間停止
2. カフェインを12時以降ゼロに変更
3. 改善しなければ [P06 光・概日リズム](./p06-light.md) を見直し
