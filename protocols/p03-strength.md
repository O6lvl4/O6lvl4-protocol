# P3. 筋力プロトコル

**目的**: 姿勢保持・代謝・骨密度を支えるインフラ筋を維持する。**筋肥大は目的にしない**。

## 不変条件

- 週2〜3回、各40分以内
- 限界の2レップ手前で止める（RPE ≤ 8）
- 同一部位は48時間以上空ける
- 知的労働の前に高ボリューム筋トレを入れない

## パラメータ

- セット数: 各種目2〜3セット
- レップ数: 6〜12
- セット間休息: 90〜120秒
- 種目数: 1セッション4〜5種目

## 種目セット

- **A日（押す＋下半身）**: スクワット / ベンチ or 腕立て / ショルダープレス / プランク
- **B日（引く＋ヒンジ＋体幹）**: デッドリフト軽め / 懸垂 or 斜め懸垂 / ローイング / デッドバグ

## 実行手順

1. ウォームアップ5分（軽い有酸素＋関節モビリティ）
2. メイン種目から重い順に実施
3. 各セット、フォーム崩壊を感じたら即終了
4. クールダウン3分（呼吸・ストレッチ）

## 違反検知

- 翌日の集中時間が著しく低下
- 関節痛・腰痛の発生
- 同重量で2週連続レップ数低下

## 復旧手順

1. 1週間ボリューム50%カット
2. 痛み発生時はその種目を2週間停止し代替種目へ
