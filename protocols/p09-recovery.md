# P09. 回復プロトコル

**目的**: 慢性疲労を蓄積させず、長期的な認知パフォーマンスを保つ。

## 不変条件

- 完全な仕事オフ日 ≥ 週1日（思考タスクなし）
- 連続労働日数 ≤ 6日
- 月1回の1日完全静養日

## パラメータ

- 平日の労働時間: ≤ 10時間
- 週末の労働時間: ≤ 4時間（理想0）
- 休暇: 四半期ごとに3〜5日

## 実行手順

1. 日曜日は仕事の Slack・メールを開かない
2. 週次レビューは金曜午後に実施し、月曜の不安を消す
3. 疲労兆候が出たら計画的に半休を取る

## 違反検知

- 朝の HRV が2週連続でベースライン-15%
- 休日に休んだ感覚が得られない
- 週次レビューで「集中できた時間」が3週連続減少

## 復旧手順

1. 即座に2日連続オフを確保
2. 運動強度を1週間半減
3. 改善しなければ睡眠を9時間に延長して1週間
