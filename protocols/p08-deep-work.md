# P8. 認知ワークプロトコル

**目的**: 1日に深い集中時間を最大化する。

## 不変条件

- 最重要タスクは午前の同じ時間帯に固定
- ディープワーク中は通知全遮断
- 1セッション ≤ 90分、間に ≥ 15分の休憩
- ディープワーク中の意思決定は仕事のみ（昼食何にするか等は前夜に決定）

## パラメータ

- ディープワーク帯1: 09:00〜12:00
- ディープワーク帯2: 13:00〜15:30
- セッション長: 50〜90分
- 休憩: 10〜20分（散歩・ストレッチ・水分）

## 実行手順

1. 前夜に翌日の最重要タスク1つを決定
2. 朝のシャローワークは09:00で打ち切る
3. ディープワーク中は Slack・メール・SNS をアプリ単位でブロック
4. 休憩中は意図的に画面を見ない

## 違反検知

- 1日の深い集中時間 < 3時間が週3日以上
- 会議・通知でディープワーク帯が侵食された日が週2日以上

## 復旧手順

1. 会議は午後に集約、午前ブロックを死守
2. 通知遮断ツールを強制化
