# P12. 口腔ケアプロトコル

**目的**: 慢性炎症を抑え、長期の認知機能低下を予防する。短期の口腔不快感も排除する。

## なぜ知的労働に効くか

歯周病による慢性低度炎症（IL-6、CRP 上昇）は、神経炎症経路を通じて認知機能低下・アルツハイマー型認知症のリスクを上げることが繰り返し報告されている。**10年単位で効く認知保護投資**。

短期的にも、口内の不快感・痛みは集中力を地味に削る。

## 不変条件

- 1日2回以上のブラッシング（朝・夜）
- 1日1回のフロス
- 半年に1回の歯科検診
- 食後30分以内の歯磨きを避ける（酸蝕症予防）
- 夜のケアは就寝60分前までに完了

## パラメータ

| 項目 | 頻度 | 内容 |
|---|---|---|
| ブラッシング | 1日2回 | 各2分以上、フッ素入り歯磨き粉 |
| フロス | 1日1回（夜） | 全歯間、ブラッシング前に実施 |
| 歯間ブラシ | 任意 | フロスが入りにくい部位 |
| 歯科検診 | 6ヶ月ごと | 症状が出てからでは遅い |
| 歯石除去 | 6〜12ヶ月ごと | 検診と同タイミング |

## 実行手順

### 朝

1. 朝食30分後にブラッシング2分

### 夜

1. 21:30 フロスで全歯間清掃
2. ブラッシング2分
3. [P11 入浴](./p11-bathing.md) 前後のルーティンに組み込む
4. 就寝60分前までに完了させ、入眠への覚醒影響を残さない

### 定期

- 半年ごとの歯科検診をカレンダーに固定登録
- 違和感が出たら次の検診を待たず即予約

## 違反検知

- 歯肉出血・口臭の慢性化 → 歯周病の可能性
- 検診間隔が9ヶ月超え
- フロスが週3日以下
- 冷たいもの・熱いものでしみる症状の発生

## 復旧手順

1. 歯科を1週間以内に予約（症状の有無問わず）
2. フロスを毎晩のルーティンに固定（ブラッシングの前）
3. 改善しなければ電動歯ブラシ＋歯間ブラシ併用に切り替え
4. 歯周病が確定したら歯科の処置スケジュールを優先する（[P9 回復](./p09-recovery.md) の休暇枠を充てる）
